Применяется элемент input, в котором обязательно указывается значение атрибута list,
которое затем применяется в id вложенного элемента datalist.
Таким образом устанавливается связь между текстовым полем и опциями
Вводя текст в поле, происходит автоматическая фильтрация доступных для выбора вариантов

Индикатор измерения значения в заданном диапазоне

Применяется для вывода значений и отображения прогресса в каком-либо известном диапазоне.
Цвет и заполненность вычисляются автоматически, на основании полученных значений

Атрибуты:
max и min - определяют максимальное и минимальное значения соответственно
value - величина значения
low и higt - поределяют пределы, при достижении которых значение считается низким и высоким соответственно
