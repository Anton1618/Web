Видео и аудио элементы

Тег audio

Подключение одного файла возможно напрямую в теге
Подключение множества файлов, возможно через вложенный элемент source

Атрибут controls - применяется для включения нативной браузерной поддержки

Подключение одного файла без применения нативной поддержки

Подключение нескольких файлов без применения нативной поддержки

Подключение нескольких файлов с примением нативной поддержки

Тег video

Подключение одного файла возможно напрямую в теге
Подключение множества файлов, возможно через вложенный элемент source

Поддерживает атрибуты:
width - задание ширины видео, height - задание высоты видео
controls - включение нативной браузерной поддержки

Подключение одного файла, без применения нативной поддержки и атрибутов ширины\высоты видео

Подключение нескольких файлов, без применения нативной поддержки и атрибутов ширины\высоты видео

Подключение нескольких файлов, без применения нативной поддержки, но с указанием ширины и высоты видео

Подключение нескольких файлов, с применением нативной поддержки, но без указания ширины\высоты видео

Подключение нескольких файлов, с применением нативной поддержки и указанием ширины\высоты видео

С включением атрибута loop для зацикливания видео

С включением атрибута autoplay для автовоспроизведения

С включением атрибута muted, для отключения звука

С включением атрибута poster, для отображения изображения во время загрузки или в результате недоступности видео файла
