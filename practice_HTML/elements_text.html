Теги code и pre

Применяются для форматирования исходного текста с заданием акцента на типе контента.

Тег code

Служит для выделения фрагментов программного кода. Отображает текст моноширинным шрифтом.

Моноширинный шрифт (monospace) — это такой тип шрифта, который соблюдает фиксированную ширину символов, используя при этом одинаковое количество горизонтального пространства для каждой буквы. Благодаря этому шрифт выглядит скорее более строгим, чем креативным. Поскольку большинство моноширинных шрифтов обычно имеют упрощенный вид, дизайнеры стараются избегать их в своих проектах.
Тем не менее, непропорциональные шрифты можно отлично вписать в различные дизайн-проекты. Такие шрифты, благодаря своему минималистичному дизайну, могут добавить профессиональный вид практически любому типу дизайна, будь то дизайн веб-сайта или плаката

const args = { name: "Yauhen", age: 30 }; const getDate = () => new Date;
Тег pre

Позволяет вывести текст на экран, сохранив изначальное форматирование. Пробелы и переносы строк при этом не удаляются

            const args = {
                name: "Yauhen",
                age: 30
            };
            const getDate = () => new Date;
