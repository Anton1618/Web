Тег input. Значения атрибута type

type - основной атрибут тега input, если этот атрибут не указан, то по умолчанию используется text

Значение text

Однострочное поле для ввода любых символов

Значение nubmer

Однострочное поле для ввода только числовых значений


Атрибуты min и max - установленные минимальное и максимальное возможные значения для ввода

Атрибут step - установленное значение шага, которое позволяет переключать введенное значение на определенную величину, по умолчанию равную единице


Значение password

Однострочное поле для ввода любых символов, значение которых будет скрыто "звёздочками"

Значение range

ползунок, для установки значения

Атрибуты используют значения по умолчанию:
min: 0
max: 100
value: min + (max-min)/2, или min if max is less than min
step: 1

Значение date

Элемент управления для ввода даты (год, месяц и день, без времени)

Значение color

Элемент управления цветом

Значение кнопок submit и reset

Изначально не имеют предопределенного поведения
Но даже без заданных свойств, имеют характерные надписи



Текстом кнопки является значение, установленное в атрибуте value



Список опций

Значение text и атрибут list со значением list

Предоставляет выпадающий список предопределенных значений элемента datalist

Значение email

Однострочное поле для ввода адреса электронной почты

Значение url

Однострочное поле для ввода url-адреса. Схож с email полем

Значение image

Кнопка-изображение для отправки формы

Значение tel

Однострочное поле для ввода телефонного номера

Значение search

Однострочное поле для ввода строки поиска

Значение datatime-local

Элемент управления для ввода даты и времени без часового пояса, потому как применяет локаль текущего устройства

Значение month

Элемент управления для ввода месяца и года без часового пояса.

Значение time

Элемент управления для ввода значения времени без часового пояса.

Значение week

Элемент управления для ввода даты, содержащей число неделя-год и номер недели без часового пояса.

Значение hidden

Скрытый для отображения на странице элемент.
Присутствие в разметке, обусловлено необходимостью внутренней обработки каких-то данных, участие в скриптах или трекинге, после чего данные отправляются на сервер

Значение file

Элемент управления, для передачи пользователем на сервер файлов, по умолчанию только одного файла


Флаг-атрибут multiple - позволяет выбирать и загружать множество файлов


Атрибут accept - задание значения типа файла, который может быть доступен для выбора и загрузки


Значение checkbox - квадратная кнопка-чекбокс

Предназначен для выбора одной, нескольких или всех предложенных для выбора опций
value - внутреннее значение кнопки, которое будет отдано этим элементом
indeterminate - указание на то, что флажок находится в неопределённом состоянии (на большинстве платформ при этом рисуется горизонтальная линия поперёк флажка).




Флаг checked - предустановленная отметка на элементе




Текстом кнопки является значение, установленное в элементе label, в который также входит и сам маркируемый элемент
Предустановлена отметка на элементе



Значение radio - круглая кнопка-переключатель

Круглая кнопка-переключатель. Предназначен для выбора только одной из всех предложенных для выбора опций
Чтобы выбранным мог быть только один элемент, а не несколько как чекбокс, должен быть задан атрибут name, а его значением должен стать повторяющийся идентификатор value для всех представленных для выбора элементов
value - внутреннее значение кнопки, которое будет отдано этим элементом




Флаг checked - предустановленная отметка на элементе




Текстом кнопки является значение, установленное в элементе label, в который также входит и сам маркируемый элемент
Предустановлена отметка на элементе
