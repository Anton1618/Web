Lists

Каждый список представляет собой контейнер, внутри которого располагаются элементы списка или пары термин-определение
Тег li - Элемент списка - List Item (li). Применяется как отдельный и самостоятельный элемент в конструкциях списков
Тег ul - Ненумерованный список - Unordered List (ul). Применяется для маркировки элементов маркерами (буллетами)
