Способы определения метки для формы

-- Может устанавливаться связь между соседними элементами метки и формы
- Для метки определяется атрибут for, для которого устанавливается определенное значение
- Для поля заполнения определяется элемент input, с атрибутом id, значение которого соответствует значению label


-- Внутри элемента label размещается
- строка со значением метки, - а также элемент формы input, с атрибутом type и значением соответствующим строке, которую оборачивает label

Тег textarea

row и cols - Обязательные атрибуты, определяющие изначальное количество строк и столбцов в поле для ввода


name - Универсальный атрибут для установки имени поля


autofocus - Автоматически производит фокус на то поле, на которое он был добавлен,
при добавлении на несколько полей - сфокусируется только первое


desabled - "Неактивное" для выбора поле, которое применяется только для чтения без возможности ввода. Отражается серым оттенком


readonly - Схож с desabled, поле может быть выбрано, но данные в нем изменить нельзя


placeholder - Добавляет текст-заменитель, который нельзя выбрать и который исчезает, при начале ввода в подобном поле


maxlength - Устанавливает ограниченное число символов для ввода


required - Устанавливает условие для обязательного заполнения поля
