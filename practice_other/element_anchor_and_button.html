Кнопки

Кнопки не представляют собой самостоятельный объект, по сравнению с ссылками.

Атрибуты

Type:
button - не имеет поведения по умолчанию, чаще всего применяется для реализации логики, например, на javascript;
submit - отправляет данные формы;
reset - очищает ввод, сбрасывает элементы управления к первоначальным настройкам;

autofocus - устанавливает автофокус на кнопке, при загрузке страницы, после этого она может быть сразу нажата, например, применяя пробел

disabled - блокирует применение кнопки для невозможности осуществления взаимодействия.
