Эквивалентно, с направлением, заданным в ручную
background-image: linear-gradient(to bottom, red, blue)

Радиальный градиент
background-image: radial-gradient(red, blue)

Эквивалентно, с направлением, заданным в ручную

Радиальный градиент

Поддерживает настройку формы

ellipse - Эллипс (по умолчанию)

background-image: radial-gradient(ellipse, red, blue)

circle - Круг

background-image: radial-gradient(circle, red, blue)

Поддерживает настройку позиционирования

(по умолчанию)
background-image: radial-gradient(circle farthest-corner, red, blue)
распространяется до дальнего угла контейнера. При изменении размеров контейнера, градиент обрезается по краям
background-image: radial-gradient(circle closest-side, red, blue)
распространение до ближайшей стороны контейнера. Градиент полностью входит в контейнер. Отталкивание происходит от ближайшей стороны
background-image: radial-gradient(circle closest-corner, red, blue)
распространение до ближайшего угла контейнера. Поэтому, он визуально больше аналогичных. При изменении размеров контейнера, градиент обрезается по краям
background-image: radial-gradient(circle farthest-side, red, blue)
распространение до дальней стороны контейнера. При изменениии размеров контейнера, градиент обрезается по краям

Применение опорных точек цвета

Применение опорных точек для указания значений распространения, приводит к более жестким переходам

Простой линейный градиент
background-image: linear-gradient(red, blue, black)
С указанием значений распространения
background-image: linear-gradient(red 25%, blue 50%, black 75%)
Простой радиальный градиент
background-image: radial-gradient(red, blue, black);
С указанием значений распространения
background-image: radial-gradient(red 25%, blue 50%, black 75%);
Простой повторяющийся линейный градиент
background-image: repeating-linear-gradient(45deg, red, blue, black)
С указанием значений распространения
background-image: repeating-linear-gradient(45deg, red 25px, blue 50px, black 75px)

Смешение цветов

Распределение пространства между цветами происходит автоматически

background-image: linear-gradient(to top right, red, blue, green, orange)

Для ручного управления пространством градиента, цветам могут быть заданы опорные точки

background-image: linear-gradient(45deg, red 0%, blue 25%, green 75%, orange 100%)

Переход между цветами может быть резким, если следующий цвет будет повторно размещен на опорной точке

background-image: linear-gradient(45deg, yellow 25%, black 25%, black 50%, yellow 50%, yellow 75%, black 75%)

Применяя повторяющийся градиент, можно получить соответствующий узор цветов
Опорные точки в процентах заменены на точные значения в px, задавая таким образом часть градиента - узор, который в дальнейшем станет повторяться

background-image: repeating-linear-gradient(45deg, yellow 25px, black 25px, black 50px, yellow 50px, yellow 75px, black 75px)
