Типы селекторов

Селектор по всем элементам

        * {
            margin: 0;
            padding: 0;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Обнуление свойств margin (отступы) и padding (поля) для ВСЕХ элементов на странице

Селектор с наследованием для применения только ко всем вложенным элементам

        * div {
            color: brown;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Свойства заданы для вложенных элементов любого блока div, но переопределены другими селекторами

Селектор по тегу

        p {
            font-family: monospace;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для всех элементов <p> на странице, задается шрифт monospace

Селектор по классу

        .ExampleClass1 {
            color: green;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для всех элементов входящих в класс ExampleClass1, задается зеленый цвет

Селектор по тегу с классом

        p.ExampleClass2 {
            color: deepskyblue;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для всех элементов с именем <p> и входящих в класс ExampleClass2, задается глубокий синий цвет

Селектор по мультиклассу

        small {
            color: rgb(211, 129, 23);
        }
        .text {
            font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
        }
        .decor {
            background-color: aquamarine;
        }
Элемент <small>
Элемент <small>
Элемент <small>

Для элементов с именем <small> задается коричневый цвет
При этом, мультиклассовость позволяет задавать дополнительные свойства Так, если элемент элементы входят в состав класса text, то к их свойствам добавляется набор шрифтов
И другие дополнительные свойства, если элемент соотетствует селектору, например, бирюзовый цвет фона текста

Селектор по атрибуту

Первый элемент <p>.
Вложенный элемент <i>.

Селектор по определенному тегу с заданным атрибутом

            color: rgb(195, 0, 255);

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Селектор по определенному тегу с заданным атрибутом для которого задано определенное значение

        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для всех элементов <p> с атрибутом title и определенным значением NewExampleTitle, задается темно-фиолетовый цвет

Второй элемент <em>
