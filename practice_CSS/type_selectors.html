Селекторы по тегу, по классу, по тегу с классом, по мультиклассу

- Селектор по тегу выбирает все элементы заданного тега на странице, независимо от их классов или других атрибутов.
- Селектор по классу выбирает элементы, которые имеют указанный класс.
- Селектор по тегу с классом позволяет выбирать элементы, которые сочетают заданный тег и класс. Это может быть полезно, если вы хотите стилизовать только определенный тип элементов с определенным классом.
- Селектор по мультиклассу позволяет выбирать элементы, которые имеют все указанные классы одновременно. Это полезно, когда нужно выбрать элементы, которые имеют конкретное сочетание классов.
- селекторы могут сочетаться и комбинироваться для более точного выбора элементов или применения стилей

        small {
            color: rgb(211, 129, 23);
        }
        .text {
            font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
        }
        small.decor {
            background-color: aquamarine;
        }
        .text.decor {
            margin-left: 10px;
        }
Применяется селектором по тегу <small> без класса. Стилизован цветом
Применяется селектором по классу .text, в данном случае, для элемента <small>. Стилизован цветом и шрифтом
Применяется селектором по тегу <small> с классом .decor. Стилизован цветом и фоном
Применяется селектором по мультиклассам .text и .decor, в данном случае, для элемента <small>. Стилизован цветом, фоном, шрифтом и отступом

Селектор по id

Обычно применяемый не для стилей, а для скриптов
Для определенного элемента с тегом IdentID1, задается розовый цвет

        #IdentID1 {
            color: pink;
        }

Селектор по атрибуту, по тегу с атрибутом, по тегу с атрибутом и значением

Для элементов, располагающих тегом title, задается светло-фиолетовый цвет
Для элементов <p> с атрибутом translate, задается фиолетовый цвет
Для элементов <p> с атрибутом title и определенным значением ExampleTitle2, задается темно-фиолетовый цвет

        [title] {
            color: rgb(226, 160, 243);
        }
        p[translate] {
            color: rgb(195, 0, 255);
        }
        p[title="ExampleTitle2"] {
            color: rgb(78, 30, 209);
        }

Применяется селектором по атрибуту title, которому бедет задан светло-фиолетовый цвет

Применяется селектором по тегу <p>, с заданным атрибутом translate, которому будет задан фиолетовый цвет

Применяется селектором по тегу <p>, с заданным атрибутом title и установленным значением ExampleClass2, которому будет задан темно-фиолетовый цвет

Селектор потомка (контекстный селектор), для вложенного элемента на любом уровне вложенности

Для элементов <p> вложенных в <div> задается розовый цвет, при этом не имеет значения уровень вложенности,
Что позволяет обращаться к определенным элементам на любой глубине от родителя и применять к ним дальнейшие инструкции

        .container1 p {
            color: pink;
        }

Селектор по первому следующему тегу после начального

применяется свойство красного цвета

        .containerA dl dt + dd {
            color: red;
        }
