Типы селекторов

Универсальный селектор для стилизации ВСЕХ элементов документа

        * {
            margin: 0;
            padding: 0;
        }
Обнуление свойств margin (отступы) и padding (поля) для ВСЕХ элементов на странице

Универсальный селектор для стилизации только по вложенным элементам

        * section {
            color: rgb(0, 141, 160);
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для всех вложенных элементов любого блока section задается темно-голубой цвет, но свойства могут быть переопределены другими селекторами

Групповой составной селектор

        h5, h6, span {
            background-color: aqua;
        }
Пример текста элемента <h5>
Пример текста элемента <h6>
Пример текста элемента <span>

Вложенный в <span> текст
Для всех указанных элементов задается свойство

Селекторы по тегу, по классу, по тегу с классом, по мультиклассу

        small {
            color: rgb(211, 129, 23);
        }
        .text {
            font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
        }
        small.decor {
            background-color: aquamarine;
        }
        .text.decor {
            margin-left: 10px;
        }
Селектор по тегу. Элемент <small> стилизован цветом
Селектор по классу. Элемент <small> стилизован цветом и шрифтом
Селектор тегу с классом. Элемент <small> стилизован цветом и фоном
Селектор по мультиклассу. Элемент <small> стилизован цветом, фоном, шрифтом и отступом

Для всех элементов <small> задается коричневый цвет
При этом, гибкость элементов позволяет задавать им дополнительные свойства, при соответствии нескольким селекторам
Так, например, если элемент соответствует нескольким селекторам, то к свойствам элемента станут добавлены правила каждого из них


Селектор по id

        #IdentID1 {
            color: pink;
        }

Первый элемент <p>.
Вложенный элемент <i>.
Следующий вложенный элемент <tt>

Второй элемент <em>

Для определенного элемента с тегом IdentID1, задается розовый цвет

Селектор по атрибуту, по тегу с атрибутом, по тегу с атрибутом и значением

        [title] {
            color: rgb(226, 160, 243);
        }
        p[translate] {
            color: rgb(195, 0, 255);
        }
        p[title="ExampleTitle2"] {
            color: rgb(78, 30, 209);
        }

Текст любого элемента с заданным атрибутом title, которому бедет задан светло-фиолетовый цвет

Текст элемента <p>, с заданным атрибутом translate, которому будет задан фиолетовый цвет

Текст элемента <p>, с заданным атрибутом title и установленным значением ExampleClass2, которому будет задан темно-фиолетовый цвет



Для элементов, располагающих тегом title, задается светло-фиолетовый цвет
Для элементов <p> с атрибутом translate, задается фиолетовый цвет
Для элементов <p> с атрибутом title и определенным значением ExampleTitle2, задается темно-фиолетовый цвет

Селектор потомка (контекстный селектор), для вложенного элемента на любом уровне вложенности

        .container1 p {
            color: pink;
        }

Текст первого уровня вложенности с применением стилей.
Вложенный текст, наследующий стили от родителя

Текст второго уровня вложенности с применением стилей.

Внешний текст, не входящий в контейнер, к которому не применяются стили



Для элементов <p> вложенных в <div> задается розовый цвет, при этом не имеет значения уровень вложенности,
Что позволяет обращаться к определенным элементам на любой глубине от родителя и применять к ним дальнейшие инструкции

Селектор первого прямого потомка, для вложенного элемента на первом уровне вложенности

        .container2 > p {
            color: pink;
        }

Текст первого уровня вложенности с применением стиля.
Вложенный текст, наследующий стили от родителя

Текст второго уровня вложенности, для которого стили не применяются.

Внешний текст, не входящий в контейнер, к которому не применяются стили



Для элементов <p> вложенных в &ldiv> задается розовый цвет,
при этом определение правил осуществляется только на первом уровне вложенности
