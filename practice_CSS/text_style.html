text-align

Свойство определяет выравнивание текста внутри элемента. Значения, которые можно использовать:
- left (выравнивание по левому краю);
- right (выравнивание по правому краю);
- center (выравнивание по центру);
- justify (выравнивание по ширине контейнера)

center

Центрирование текста по горизонтали


left

Выравнивание текста по левому краю


right

Выравнивание текста по правому краю


justify

Контейнер <div>, имеющий ширину в 500px и расположенный по центру автоматически
Следует заметить, что justify работает только в рамках блочных элементов, таких как <p> или <div>. Он не будет работать на строчных элементах, таких как <span> или <a>. Также обратите внимание, что в некоторых случаях есть ограничения на использование этого свойства, особенно при работе с другими CSS-свойствами, такими как text-indent или text-justify

unicode-bidi

Свойство используется для управления двунаправленным текстом в элементе. Оно может быть полезно при работе с многоязычными текстами. Применяемые значения:
- normal (стандартное направление);
- embed (внутри элемента используется другое направление);
- bidi-override (задает направление текста явно).

normal

По умолчанию, normal - означает, что направление текста будет определено языком по умолчанию для элемента

Lorem ipsum dolor


embed

Текст будет "встроен" в элемент, что означает, что его направление будет определяться на основе других внешних контекстов, таких как родительские элементы или стили
Демонстрируется применение свойства direction, для явного указания направления текста. В первом абзаце указан direction: ltr, чтобы указать левостороннее направление текста (LTR), а во втором абзаце указан direction: rtl, чтобы указать правостороннее направление текста (RTL)
