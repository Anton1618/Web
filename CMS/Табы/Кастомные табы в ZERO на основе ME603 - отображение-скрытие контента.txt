В Zero cоздаются декоративные элементы, которые станут выполнять дальнейший функционал табов me603 

- Создается zero блок с кастомными вкладками для переключения.
Лучше оставлять минимум места до следующего блока. 
Вкладки могут быть:
	- шейпами, если необходима последующая загрузка фото
	- простым фото, если необходимо сохранение пропорций при адаптиве
	- кнопкой, если необходимо просто задать красивый текст и расположение элемента
Для всех текстов на карточках устанавливается настройка Pointer Events: None, для игнорирования текста при клике (по блоку вкладки)

- Создаются блоки заголовков (например в zero с простым текстом)

- Создаются блоки карточек товаров для каждого переключаемого заголовка

- Создается блок ME603 с переключаемыми табами и заполняется данными для переключения блоков
	- В соответствующих полях прописываюся id блоков, необходимых для отображения этой вкладкой.
Копируется адрес каждого таба вкладки блока me603
	- Каждый таб имеет продолжение от 1 до 8 после id блока, можно просто скопировать в адресной строке название каждой вкладки таба с её номером, начиная со слеша. Так, например, таб первой вкладки станет иметь следующий вид: #!/tab/612330148-1
ID каждого таба последовательно вставляется в соответствующие URL каждой кастомной вкадки ранее созданного zero блока

- После проделанных операций, клик по кастомным табам, дает переключение отображения блоков, аналогично стандартному блоку me603. Сам блок скрывается, оставляя на странице только его функционал
