https://goodweb.studio/zero-block-accordeon#info1

- Создаем кнопку в нужном нам месте и указываем ей ссылку типа #name. Первая кнопка #name1, вторая #name2 и т.д.;
- Создаем блок, который будет скрыт и будет открываться по клику на наш аккордеон, наполняем его контентом;
- Добавляем код и меняем в нем номер блока (#rec в двух местах), который будет скрыт;
- Повторяем процедуру для каждого блока заголовок-контент

=========================== Для каждой пары заголовок-контент ================================
<style>
/*Скрываем наши блоки по ID*/
#rec321583301  {
    display:none;
}
</style>

<script>
//При нажатии на кнопку со ссылкой #name
 $('[href = "#name"]').click(function() {
//Открываем первый блок 
   $('#rec321583301').slideToggle(500);
   t_lazyload_update();
});
</script>
