Переход между блоками, благодаря zero с опцией overflow: visible
В блоке размещается элемент, смещенный в сторону перекрываемого блока, для создания эффекта переключения. 
	- Элементу задается 100% ширины по window контейнеру
	- Устанавливается класс для обращения, например, shape-transform
	- В step by step анимании задается анимация по скролу, по Start Trigger - Windows Buttom или Window Center. Первый шаг устанавливается для уменьшения элемента по оси y до значения 0, может потребоваться установка задержки Delay, например, для установки триггера с центра с задержкой в 300 пикселей
	- Элемент устанавливается со смещением на следующий блок примерно на половину своей высоты

Важным условием приведенного эффекта на примере является задание процентного скругления для этого элемента, что в самой тильде недоступно
В инспекторе находим элемент и задаем ему

		border-radius: 70% !important;
	}
